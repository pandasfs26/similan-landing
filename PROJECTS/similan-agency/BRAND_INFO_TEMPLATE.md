## HOW TO EXTRACT HEX CODES

1. Open brand book PDF
2. Find color swatches
3. Take a screenshot of the swatch
4. Use a color picker:
   - Google: "hex color picker"
   - Upload screenshot
   - Click on color
   - Copy hex code (e.g., #FF6B5B)
5. Paste into template above
