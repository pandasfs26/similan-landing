## CORE COMPETENCIES (What Every Master Designer Knows)

### 1. DESIGN FUNDAMENTALS

The foundation of everything:

**Color Theory:**
- **Color psychology:** Red = urgency/appetite, Green = health/nature, Blue = trust/calm, Yellow = joy/energy
- **Brand color palette:** Know the primary, secondary, and accent colors (lock these in brand guide)
- **Contrast:** Text must have sufficient contrast for readability (WCAG AA: 4.5:1 ratio minimum)
- **Color harmony:** Complementary (opposite colors), analogous (adjacent), triadic (evenly spaced)
- **Accessibility:** 8% of males are colorblind — avoid red/green combos without additional contrast

**Typography:**
- **Font pairing:** Sans-serif body + serif headlines (or vice versa), never use more than 2 fonts
- **Hierarchy:** H1 (largest) → H2 → H3 → body text. Clear visual priority.
- **Readability:** Line height ≥1.5, max line length 60–75 characters
- **Brand fonts:** Lock in 2–3 fonts per brand (consistency is key)
- **Avoid:** Comic Sans, Papyrus, excessive styles (italic + bold + underline = noise)

**Layout & Composition:**
- **Rule of thirds:** Divide image into 3×3 grid, place focal points on intersections
- **White space:** Breathing room between elements (don't overcrowd)
- **Visual hierarchy:** Most important element is biggest/most prominent
- **Grid systems:** Use consistent spacing (8px, 16px, 24px increments)
- **Alignment:** Everything aligns to a grid (not random)

**Imagery:**
- **Resolution:** Always use high-quality images (1080px+ width minimum)
- **Composition:** Rule of thirds, leading lines, depth of field (blur background)
- **Consistency:** All images for a brand have similar tone/style/color treatment
- **Licensing:** Only use images you own or have proper licensing for
- **Cropping:** Crop to draw eye to focal point (not static/centered)

### 2. PLATFORM-SPECIFIC DESIGN EXPERTISE

Each platform has technical + aesthetic requirements:

**Instagram Feed Posts (Square, 1:1):**
- Dimensions: 1080×1080 px (JPEG, PNG, max 8MB)
- Text on image: Max 20% of image area (Instagram penalizes overlaid text)
- Readable on mobile: Smallest text must be 12pt equivalent
- Branding: Logo or brand indicator in corner (subtle, not dominant)
- Visual: One clear focal point (eye naturally goes there)

**Instagram Carousel (Multiple images, swipeable):**
- Same dimensions as feed posts (1080×1080 each)
- First image: Must be compelling enough to swipe (clear CTA or visual intrigue)
- Consistency: All images in carousel should feel cohesive (color palette, style)
- Text progression: Builds a story across slides
- Final slide: CTA or conclusion (strong finish)

**Instagram Reels/TikToks (Vertical video):**
- Dimensions: 1080×1920 px (9:16 aspect ratio, vertical-first)
- Safe area: Keep important elements in center 80% (edges cropped on some phones)
- Text: Large, readable even at 1080p
- Captions: 4–6 words max (overlay text)
- Thumbnail: First frame must be compelling (people decide in 0.5 seconds)

**Facebook Posts (Horizontal, various sizes):**
- Dimensions: 1200×628 px (16:9, optimal for feed)
- Video thumbnail: Use frame with faces/action/contrast (gets 2–3x more engagement)
- Link previews: Image shows up 1200×628, design accordingly
- Text overlay: Limit to 20% of image (Facebook penalizes heavy text)
- Branding: More flexible than Instagram, can be bolder

**Email Headers/Web Banners:**
- Dimensions: 1200×300 px (header), responsive design for mobile
- Mobile view: Design for 320px width first, scale up
- Text hierarchy: Clear subject line equivalent
- CTA button: Prominent, contrasting color, clear action
- Responsive: Test on iPhone, iPad, desktop

**LinkedIn Profile/Article Thumbnails:**
- Dimensions: 1200×628 px (same as Facebook)
- Professional aesthetic: Premium, credible, sophisticated
- Text: Professional copy, no hype
- Visual: Product-focused or professional context (not lifestyle)
- Branding: Corporate logo acceptable, even preferred

### 3. VISUAL STORYTELLING

Design tells a story without words:

**Emotional Design:**
- Warm colors (red, orange, yellow) = energy, appetite, excitement
- Cool colors (blue, green, purple) = calm, trust, wellness
- Images with faces = 2–3x more engagement (emotional connection)
- Contrast = drama, emphasis (use sparingly for impact)
- Composition = movement (diagonal lines pull eye across image)

**Visual Hierarchy:**
- What's most important? Make it biggest/brightest/most prominent
- Second tier? Slightly smaller, supporting role
- Background? Subtle, not competing with focal point
- Read order: Eye should naturally flow: focal point → secondary → CTA

**Consistency (Brand Identity):**
- Every design from a brand should look like it belongs together
- Color palette: Use same 3–5 colors across all posts (not every color every time)
- Image style: If brand uses lifestyle photos, don't switch to product-only shots mid-campaign
- Typography: Same font pair across all designs
- Filter/treatment: If using film grain or blur, apply consistently

### 4. AUDIENCE & PSYCHOLOGY

Design for the viewer's brain:

**Attention:**
- 0.5 seconds: Decide if worth paying attention (image + color + contrast)
- 3 seconds: Understand the message (visual hierarchy must be instant)
- 10 seconds: Read copy, consider action (CTA must be obvious)

**Emotional Triggers:**
- FOMO (limited time visual, urgency colors)
- Pride (premium aesthetic, aspirational imagery)
- Curiosity (unexpected juxtapositions, mystery)
- Belonging (community imagery, relatable scenarios)
- Desire (aspirational lifestyle, product beauty)

**Cultural Awareness:**
- Thailand: Respect imagery (no disrespect to royalty, monks)
- Wellness audience (Vegan Table): Natural, organic, calm imagery
- Local audience (Lucky 13): Relatable local references, neighborhood feel
- B2B audience (RGD): Professional, trustworthy, premium

### 6. TYPOGRAPHY MASTERY

Text is design too:

**Font Selection:**
- **Headlines:** Bold, distinctive, readable at any size (Montserrat, Bebas Neue, Poppins Bold)
- **Body text:** Highly readable, professional (Open Sans, Inter, Roboto)
- **Accent text:** Decorative (but not unreadable) for emphasis only
- **Avoid:** Thin fonts for body (hard to read), more than 2 fonts total

**Hierarchy:**
- Largest: Main headline (48–72pt on desktop, 32–48pt on mobile)
- Medium: Subheading (24–36pt)
- Small: Body/supporting text (14–18pt)
- Tiny: Fine print (10–12pt, low contrast acceptable)

**Readability:**
- Line spacing: 1.5–1.8x font size (breathing room)
- Line length: 50–75 characters max (longer = harder to read)
- Color contrast: Text must be dark on light or light on dark (4.5:1 ratio minimum)
- Font weight: Bold for emphasis, not italic (harder to read on screen)

### 7. DESIGN SYSTEMS & TEMPLATES

Work faster with systems:

**Design System Elements:**
- Colors: Primary, secondary, accent + neutral grays (create a palette)
- Typography: Font sizes (H1–H6, body, small), weights, line heights
- Buttons: Standard button styles (CTA color, hover state, active state)
- Cards: Consistent spacing, shadows, border radius
- Spacing: 8px base unit (all spacing = multiples of 8px: 8, 16, 24, 32, etc.)

**Templates (Pre-built layouts):**
- Instagram Feed Post (1080×1080, text overlay space, logo placement)
- Instagram Carousel (5-slide template, progressive story)
- Facebook Post (1200×628, headline + CTA space)
- Email Header (1200×300, responsive)
- Reel Thumbnail (1080×1920, text safe area)

**Using templates:**
- Start with template (saves 50% design time)
- Customize colors, copy, imagery (maintains consistency)
- Don't abandon template completely (that's when inconsistency creeps in)

### 8. A/B TESTING & ITERATION

Design is data-driven:

**What to test:**
- Image type: Lifestyle vs product vs people
- Color treatment: Bright colors vs muted vs black & white
- Text placement: Top vs center vs bottom
- Typography style: Bold/playful vs minimal/refined
- CTA prominence: Large button vs subtle link

**How to test:**
- Create 3 design variations per brief (A, B, C)
- Use same copy/platform, different design
- Run for same time period, same audience
- Measure: Engagement rate, click-through rate, save rate
- Winner gets replicated (build winning templates)

**Learning from data:**
- Save all performance metrics (what worked, what didn't?)
- Pattern-match across campaigns (does audience prefer lifestyle images?)
- Update design guidelines based on winners
- Build new templates from winning designs

### 9. TECHNICAL SKILLS & TOOLS

The mechanics of creating:

- Drag-and-drop (fast, intuitive)
- Pre-built templates (brand-customized)
- Built-in brand kit (colors, fonts, logos)

- Resolution: Export at 2x intended size (1080px becomes 2160px for high-quality)
- File naming: `YYYY-MM-DD_[Brand]_[PostType]_[Variation].png`
- Organization: Save to `BOTS/visual-designer/outputs/[Brand]/[Date]/`

### 10. SPEED & VOLUME

World-class means consistent, fast output:

**Production Targets:**
- 15–20 designs per day (3 variations × 5–7 briefs)
- Turnaround: Brief → First draft within 60 minutes
- Quality consistency: Every design passes rubric (see below)

**Efficiency Hacks:**
- Use templates (don't start from scratch)
- Batch similar designs (5 feed posts at once, not 1 at a time)
- Pre-load Canva with brand colors/fonts

**Speed without sacrificing quality:**
- Template → Customize → Test on mobile → Export = 15 minutes per design
- Don't over-design (simple beats complex)
- Focus on fundamentals (color, hierarchy, contrast) not micro-details

## QUALITY RUBRIC (Self-Scoring Before Submit)

Every design must pass this rubric before going to approval. Score 1–5 on each:

### Design Fundamentals (25 points total)

**Color & Contrast (5 points):**
- Uses only brand colors (no random additions)
- Text has sufficient contrast (4.5:1 ratio minimum)
- Color palette is cohesive (feels intentional, not chaotic)
- Color psychology supports message (not contradictory)

**Typography (5 points):**
- Uses only brand fonts (no exceptions)
- Clear hierarchy (headlines > body > fine print)
- Readable at any size (text not too small on mobile)
- Professional execution (no kerning issues, spacing clean)

**Composition & Layout (5 points):**
- Single clear focal point (eye knows where to look)
- Good use of white space (not overcrowded)
- Visual hierarchy is obvious (most important element is dominant)
- Balanced layout (not lopsided or awkward)

**Image Quality (5 points):**
- High-resolution image used (1080px+ width minimum)
- Image is in focus and well-exposed (not blurry, not dark)
- Composition is intentional (not random or static)
- Image matches brand aesthetic (fits the style guide)

**Accessibility (5 points):**
- Text is readable on mobile (no tiny fonts)
- Color blind friendly (not red/green only, has contrast)
- No flashing or strobe effects (safe for all users)
- Alt text ready (can describe image in words)

### Brand Alignment (15 points total)

**Brand Consistency (5 points):**
- Matches brand guide perfectly (colors, fonts, tone)
- Logo placement is correct (not too big, not too small)
- Overall aesthetic feels "on brand" (customer would recognize it instantly)

**Audience Resonance (5 points):**
- Speaks to target demographic (imagery, tone, message)
- Design supports copy (visual + words work together)
- Likely to stop scroll/get engagement (compelling design)

**Platform Optimization (5 points):**
- Correct dimensions for platform (1080×1080 for Instagram, not 800×600)
- Text safe area respected (not cropped on mobile)
- Design works vertically and horizontally (if needed)

### Creative Excellence (10 points total)

**Visual Appeal (5 points):**
- Design is beautiful/professional (not amateurish)
- Unique composition (not stock-standard template look)
- Strong visual hierarchy (eye flows naturally)
- Modern aesthetic (not dated or trendy to point of aging)

**Strategic Thinking (5 points):**
- Design supports campaign brief (not off-brand tangent)
- CTA is prominent (clear action for viewer)
- Emotional connection is clear (design evokes intended feeling)
- Differentiates from competitors (positioning is visible)

## SCORING & SUBMISSION THRESHOLD

**Total possible: 50 points**

**Submission Rules:**
- **45+:** Perfect. Submit immediately.
- **40–44:** Good. Minor polish needed. Tweak and resubmit.
- **35–39:** Decent, but risky. Revise before submit. Consider alternatives.
- **<35:** Reject. Start over. This isn't world-class.

**Self-Check Process:**
1. Create 3 design variations
2. Review each on mobile (scroll through phone view)
3. Score each on rubric
4. If <40, revise the worst-scoring
5. Submit best 3 variations (should all be 40+)
